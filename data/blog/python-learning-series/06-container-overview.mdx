# 容器型別（Data Structures）

Python 提供了四種常用的容器型別，每種都有不同的特性與適用場景：

| 型別    | 說明       | 可變性 | 範例               |
| ------- | ---------- | ------ | ------------------ |
| `list`  | 有序可重複 | 可變   | `[1, 2, 3]`        |
| `tuple` | 有序不可變 | 不可變 | `(1, 2, 3)`        |
| `set`   | 無序唯一   | 可變   | `{1, 2, 3}`        |
| `dict`  | 鍵值對     | 可變   | `{'a': 1, 'b': 2}` |

---

## Quick overview

```python
# List
fruits = ["apple", "banana", "cherry"]

# Tuple
point = (10, 20)

# Set
unique_numbers = {1, 2, 3}

# Dict
person = {"name": "Alice", "age": 25}
```
